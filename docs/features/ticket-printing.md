# 感熱プリンター待機整理券発券 (Ticket Printing)

> 最終更新: 2026-07-10  
> 関連パッケージ: `pdf`, `printing`

## 概要

店舗スタッフが手動でお客様を登録した際、店内に設置された感熱（レシート）プリンターを介して、その場で実物の待機整理券（チケット）を出力・発券する仕組みです。

---

## 印刷までのフロー

```
スタッフが手動受付フォームから登録を実行
      │
      ▼
アプリ内部で整理券用 PDF を動的に生成 (pdf パッケージ)
      │   (店舗名、待機番号、予想待ち時間、受付人数、QRコードを含む)
      ▼
OS標準の印刷機能へPDFを転送 (printing パッケージ)
      │
      ▼
(ペアリングされたBluetooth / Wi-Fi 感熱プリンター) 印刷実行
```

---

## 主要な実装アプローチ

- **感熱紙の規格サイズへの最適化**: 58mm または 80mm の標準的なレシート用紙の横幅に収まるよう、PDF生成時に余白（Margin）やフォントの最大文字幅を厳格に計算して動的にレイアウトを構成します。
- **多言語フォントの埋め込み**: PDFレンダリング時に日本語や韓国語が文字化け（豆腐文字）する問題を防ぐため、アプリのアセットに含まれるカスタムTTFフォントをPDF文書に直接バインドして静的に読み込みます。
- **OSの抽象化による高い互換性**: 各種プリンターメーカー固有のSDKを直接組み込むのではなく、OS標準の印刷サービス（Android Print Service、iOS AirPrint）へブリッジする `printing` ライブラリを採用。これにより、店舗に既存の多種多様なプリンターモデルとプラグアンドプレイで連携可能です。
