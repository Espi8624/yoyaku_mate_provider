# 감열 프린터 티켓 출력 (Ticket Printing)

> 최종 수정: 2026-07-10  
> 관련 라이브러리: `pdf`, `printing`

## 개요

매장에서 수동으로 대기 손님을 등록할 경우, 영수증(감열) 프린터를 통해 실물 대기 번호표를 즉시 출력하는 기능입니다. 모바일 기기의 기본 프린트 스풀러(AirPrint, Android Print Service)와 연동됩니다.

---

## 출력 흐름

```
직원이 수동으로 손님 등록 완료
      │
      ▼
앱에서 영수증 형태의 PDF 문서 동적 생성 (pdf 패키지)
      │   (가게 이름, 대기 번호, 예상 대기 시간, 인원 수 포함)
      ▼
OS 인쇄 시스템으로 PDF 전달 (printing 패키지)
      │
      ▼
(블루투스/Wi-Fi 감열 프린터) 티켓 출력
```

---

## 주요 구현 포인트

- **PDF 레이아웃 최적화**: 58mm 또는 80mm 감열 영수증 용지 폭에 맞도록, 한정된 영역 안에 텍스트가 잘리지 않게 동적 레이아웃 크기를 지정합니다.
- **다국어 폰트 대응**: 한국어/일본어 등 다국어가 포함된 경우, `pdf` 패키지에서 기본 제공하지 않는 폰트 문제를 해결하기 위해 시스템 에셋에서 TTF 폰트를 바인딩하여 렌더링합니다.
- **OS 네이티브 연동**: `printing` 라이브러리를 통해 Android/iOS 운영체제의 기본 인쇄 다이얼로그를 띄워, 프린터 드라이버나 복잡한 네트워크 연동 없이 모바일에 연결된 프린터를 바로 사용합니다.
